# BookHaven — Online Book Store

BookHaven is a premium full-stack online book store application built with Python and Flask. It features a stunning, "Web 3.0" inspired dark theme UI with advanced glassmorphism, 3D CSS hover effects, and ambient background animations.

## Features

- **User Authentication:** Secure registration and login using session management and Bcrypt password hashing.
- **Dynamic Catalog:** Browse and search a curated collection of books with beautiful, animated 3D hover cards.
- **Shopping Cart:** Add books to your cart, adjust quantities, and manage items seamlessly.
- **Enhanced Checkout Flow:** Provide shipping addresses and simulated credit card validation during checkout (supports Card, UPI, and COD).
- **Automated Invoices:** Instantly generates downloadable PDF invoices for every successful order using ReportLab.
- **Analytics Dashboards:** 
  - **User Analytics:** Interactive Chart.js charts displaying your personal spending history.
  - **Admin Analytics:** Platform-wide metrics for total revenue and order volume.
- **Admin Control Panel:** Dedicated dashboard for authorized users to add/edit/delete books and manage user roles.

## Tech Stack

- **Backend:** Python 3, Flask, Flask-SQLAlchemy (SQLite), Flask-Login, Flask-Bcrypt
- **Frontend:** HTML5, modern vanilla CSS3 (CSS Variables, Grid, Keyframe Animations), and Vanilla JavaScript
- **Utilities:** ReportLab (PDF Generation), Chart.js (Data Visualization)
- **Design:** *Plus Jakarta Sans* font, custom glassmorphism, glowing ambient CSS lighting, responsive layout

## Installation & Setup

1. **Clone the repository and initialize your environment:**
   Ensure you have Python 3.9+ installed on your system.

2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Seed the Database:**
   We provide a handy seating script that sets up the database schema and populates it with 10 sample books, an admin account, and a few sample regular users.
   ```bash
   python seed.py
   ```

   **Test Accounts generated by seed data:**
   - **Admin:** `admin@bookhaven.com` / `password123`
   - **User:** `ravi@example.com` / `password123`

4. **Run the Development Server:**
   ```bash
   python run.py
   ```
   The application will be available at `http://127.0.0.1:5000/`.

## Project Structure

```text
online-bookstore/
│
├── app/
│   ├── static/             # CSS files, JS files, downloaded Invoices, and Assets
│   ├── templates/          # Jinja2 HTML templates
│   ├── __init__.py         # Application Factory
│   ├── models.py           # SQLAlchemy Database Models (User, Book, Order, etc.)
│   ├── auth.py             # Authentication Routes
│   ├── books.py            # Catalog & Search Routes
│   ├── cart.py             # Shopping Cart Routes
│   ├── payment.py          # Checkout & Payment Flow
│   ├── invoice.py          # PDF Generation Logic
│   ├── analytics.py        # Chart.js Data & View Logic
│   └── admin.py            # Admin Dashboard & CRUD Operations
│
├── instance/               # Contains the SQLite database (bookstore.db)
├── config.py               # Application configuration
├── run.py                 # Main application entry point
├── requirements.txt        # Python package dependencies
└── seed.py                 # Database initialization and sample data
```
